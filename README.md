# FastApi

*This is an example to deploy a model using FastApi*

## Installation

1. Install fastapi and uvicorn packages
   
```
pip install fastapi uvicorn
```
2. Run FastApi local host

```
python -m uvicorn main:app --reload
```

3. Make request in Postman
